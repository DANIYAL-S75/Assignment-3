# Case Counter - Python Assignment 3

A simple Python program that accepts a series of words from the user and counts the number of uppercase and lowercase letters.

## 📌 Project Description
This project was developed to satisfy the requirements of Assignment 3 for the "ML & Gen AI" course. The program takes a dynamic string input from the console and iterates through each character to determine its case, ignoring symbols and numbers.

## 🚀 Features
*   **Dynamic Input**: Accepts any string directly from the terminal.
*   **Case Sensitivity**: Accurately distinguishes between `UPPER CASE` and `LOWER CASE`.
*   **Validation**: Uses Python's built-in `.isupper()` and `.islower()` methods for reliable results.

## 🛠️ How to Run
1.  Ensure you have **Python 3.x** installed.
2.  Download the script file: `assignment_case_counter.py`.
3.  Open your terminal or command prompt.
4.  Navigate to the directory containing the file and run:
    ```bash
    python assignment_case_counter.py
    ```
